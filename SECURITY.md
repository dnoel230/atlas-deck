# Security Policy

## Supported Versions

Atlas Deck is early alpha software. Use the latest commit on `main` unless a release is published.

## Reporting a Vulnerability

Do not open a public issue with secrets, private IPs, passwords, tokens, or exploit details.

For now, report security concerns by opening a GitHub issue with a general description and no sensitive details.

## Scope

Atlas Deck is intended for:

- Personal homelabs
- Authorized systems
- Local development
- Secure remote admin

Out of scope:

- Unauthorized scanning
- Credential theft
- Bypassing access controls
- Tools that target systems without permission
