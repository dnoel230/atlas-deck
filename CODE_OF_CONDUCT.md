# Code of Conduct

Atlas Deck should be useful, safe, and welcoming.

## Expected Behavior

- Be respectful.
- Stay on topic.
- Help others learn.
- Give clear feedback.
- Assume good intent.

## Unacceptable Behavior

- Harassment
- Personal attacks
- Sharing private data
- Encouraging unauthorized access
- Publishing secrets or credentials

## Security and Ethics

Atlas Deck is for systems you own or are authorized to manage. Do not use this project to target third-party systems without permission.

## Reporting Issues

Open a GitHub issue if something needs attention. For sensitive security issues, do not post secrets or private details publicly.
